Defeitos Comuns

Código: tE, LE, AE, dE

Código tE: Este código indica algum problema na secagem.

Código LE: Se o ciclo de lavagem for interrompido e código de erro LE for exibido no painel de sua lavadora, é uma indicação que ocorreu uma falha de comunicação entre a placa de controle e o motor ou sobrecarga.

Código AE: Este código indica que houve transbordamento interno de água. Entre em contato imediatamente e agende uma visita técnica.

Código dE: Se o código DE for exibido no painel de sua lavadora, é uma indicação que ocorreu algum problema relacionado à porta.

Em alguns casos, pode aparecer:

dE1: Indica que a porta está quebrada
dE2: Indica que a porta não foi fechada corretamente.

Código: FE, PF, OE, IE

Código FE: Se houver excesso de água por uma falha na válvula de entrada, o código é exibido na tela.

Código PF: Se houver queda de energia será exibido no painel de sua lavadora.

Código OE: Se o início da drenagem não for executado dentro de 10 minutos, o código OE é exibido na tela e o problema está na mangueira de drenagem.

Código IE: Se durante a lavagem sua lavadora não encher de água em até 8 minutos, o código é exibido e está é uma indicação que ocorreu algo com o fernecimento de água.

Código: PE, Cd, FF, UE

Código PE: Se existir uma falha de funcionamento do sensor do nível da água, será exibido no painel da lavadora o código de erro PE.

Código Cd: No processo de secagem o tambor irá atingir alta temperaturas, para sua segurança e evitar que algo ocorra quando for retirar ás roupas da lavadora a porta trava automaticamente e no visor será exibido o código CD

Código FF: Este código será exibido apenas se houver congelamento da água nas mangueiras de entrada ou drenagem. Geralmente ocorre em locais muito frio.

Código UE: Quando a lavadora apresenta o código de UE, esta sinalizando que há um desequilíbrio dentro do tambor de sua lavadora.

Código: CE, dHE, 4E, 3E6

Código CE: Este código de erro representa algo relacionado ao motor.

Código dHE: Falha durante o ciclo de secagem. Está é uma indicação que algo ocorreu com algumas peças responsáveis pela secagem.

Recomendamos que solicite suporte imediato com profissional especialista.
